Detect objects and their locations in images.

## Quick Start

First, install the package:

<Tabs items={['Python', 'Node.js', 'cURL']}>
  <Tabs.Tab>
~~~bash
pip install moondream pillow
~~~
  </Tabs.Tab>
  <Tabs.Tab>
~~~bash
npm install moondream
~~~
  </Tabs.Tab>
  <Tabs.Tab>
~~~bash
# No installation needed
~~~
  </Tabs.Tab>
</Tabs>

Place an image file in your working directory, then:

<Tabs items={['Python', 'Node.js', 'cURL']}>
  <Tabs.Tab>
~~~python
import moondream as md
from PIL import Image

# Initialize with API key
model = md.vl(api_key="your-api-key")

# Load an image
image = Image.open("path/to/image.jpg")

# Detect objects (replace 'subject' with what you want to detect)
detect_result = model.detect(image, "subject")
print("Detected objects:", detect_result["objects"])
~~~
  </Tabs.Tab>
  <Tabs.Tab>
~~~typescript
import { vl } from "moondream";
import fs from "fs";

const model = new vl({ apiKey: "your-api-key" });
const image = fs.readFileSync("path/to/image.jpg");

async function main() {
  // Regular response (replace 'subject' with what you want to detect)
  const detect_result = await model.detect({
    image,
    subject: "subject"
  });
  console.log(detect_result);
}

main();
~~~
  </Tabs.Tab>
  <Tabs.Tab>
~~~bash
curl --location 'https://api.moondream.ai/v1/detect' \
--header 'X-Moondream-Auth: <API KEY FROM console.moondream.ai>' \
--header 'Content-Type: application/json' \
--data '{
    "image_url": "data:image/jpeg;base64,<BASE64-STRING>",
    "object": "<NAME OF OBJECT TO DETECT>",
    "stream": false
}'
~~~
  </Tabs.Tab>
</Tabs>
