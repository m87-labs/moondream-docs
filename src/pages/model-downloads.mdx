# Model Downloads

All Moondream models are available in different sizes and quantization formats to suit your needs.

## Understanding Quantization
- **FP16**: 16-bit floating point - highest precision, largest size
- **INT8**: 8-bit integer quantization - good balance of precision and size
- **INT4**: 4-bit integer quantization - smallest size, some precision trade-off

## Latest Models

### Moondream 0.5B
Lightweight models optimized for faster inference and lower resource usage:
- [Moondream 0.5B INT8 (622MB)](https://huggingface.co/vikhyatk/moondream2/resolve/onnx/moondream-0_5b-int8.mf.gz?download=true) - Balanced performance, better accuracy than INT4
- [Moondream 0.5B INT4 (442MB)](https://huggingface.co/vikhyatk/moondream2/resolve/onnx/moondream-0_5b-int4.mf.gz?download=true) - Most compact size, fastest loading, good for testing

### Moondream 2B 
Full-size models with higher accuracy and more capabilities:
- [Moondream 2B FP16 (2.88GB)](https://huggingface.co/vikhyatk/moondream2/resolve/client/moondream-latest-f16.bin.gz?download=true) - Full precision, highest accuracy, best for production use
- [Moondream 2B INT8 (1.83GB)](https://huggingface.co/vikhyatk/moondream2/resolve/client/moondream-latest-int8.bin.gz?download=true) - Great balance between model size and accuracy
- [Moondream 2B INT4 (1.31GB)](https://huggingface.co/vikhyatk/moondream2/resolve/client/moondream-latest-int4.bin.gz?download=true) - Smallest 2B variant, good for resource-constrained environments

<Callout type="info">
  It's good practice to create a dedicated `models` directory in your project folder and place the downloaded model file there.
</Callout>
