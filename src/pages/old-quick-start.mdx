<Steps>
### Create Project Directory
Create a new directory for your project

### Download Model
Download the model by clicking here: [Download Moondream 0.5B INT8](https://huggingface.co/vikhyatk/moondream2/resolve/onnx/moondream-0_5b-int8.mf.gz?download=true)

<Callout type="info">
  Check out other available models at [Model Downloads](/model-downloads) for different size and performance options.
</Callout>

### Move Model File
Move the downloaded file to your project directory

### Install Package
<Tabs items={['CPU', 'GPU']}>
  <Tabs.Tab>
```bash
# Run this in your project directory
pip install moondream
```
  </Tabs.Tab>
  <Tabs.Tab>
  <Callout type="info">
    PIP library for GPU support coming soon!
  </Callout>
  - Prerequisites for GPU support
      - [CUDA 12.x](https://developer.nvidia.com/cuda-downloads)
      - [cuDNN 9.x](https://developer.nvidia.com/cudnn)
```bash
# Run this in your project directory
pip install moondream-gpu
```
  </Tabs.Tab>
</Tabs>

### Run Sample Code
```python
from moondream import vl
from PIL import Image
from urllib.request import urlopen

model = vl('moondream-0_5b-int8.bin.gz')

url = 'https://upload.wikimedia.org/wikipedia/commons/thumb/4/4d/Cat_November_2010-1a.jpg/1280px-Cat_November_2010-1a.jpg'
image = Image.open(urlopen(url))

for t in model.caption(encoded_image, stream=True)['caption']:
    print(t, end='', flush=True)
# optionally, you can use the non-streaming version
# print(model.caption(encoded_image)['caption'])
```
