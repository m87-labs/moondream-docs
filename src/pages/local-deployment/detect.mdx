# Object Detection

Detect and locate objects in images with Moondream.

## Installation & Model

~~~bash
pip install moondream==0.0.2
~~~

Download the model from our [Model Downloads](/model-downloads) page.

## Basic Usage

~~~python
from moondream import VL
model = VL("models/moondream-2b-int8.bin.gz")

# Basic object detection
objects = model.query(image, "What objects do you see in this image?")
print(objects['answer'])

# Location-based detection
locations = model.query(image, "Where is the cat in this image?")
print(locations['answer'])
~~~

That's all you need to start detecting objects!
