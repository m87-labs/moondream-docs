## Overview

Moondream's object detection feature offers:
- Precise object localization
- Confidence scores for detections
- Support for multiple object classes
- Bounding box coordinates

### 3. Detect Objects
<Tabs items={['Python', 'JavaScript', 'TypeScript']}>
  <Tabs.Tab>
    ```python
    # Detect objects
    response = await api.detect_object(
        image_url="https://example.com/image.jpg",
        object="person"
    )
    
    # Process results
    for detection in response.result:
        confidence = detection['confidence']
        bbox = detection['bbox']  # [x1, y1, x2, y2]
        print(f"Found object with {confidence:.2f} confidence at {bbox}")
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```javascript
    // Detect objects
    const response = await api.detectObject({
      imageUrl: "https://example.com/image.jpg",
      object: "person"
    });
    
    // Process results
    response.result.forEach(detection => {
      const { confidence, bbox } = detection;
      console.log(`Found object with ${confidence.toFixed(2)} confidence at ${bbox}`);
    });
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```typescript
    interface Detection {
      confidence: number;
      bbox: [number, number, number, number];
    }

    // Detect objects

      imageUrl: "https://example.com/image.jpg",
      object: "person"
    });
    
    // Process results
    response.result.forEach((detection: Detection) => {
      const { confidence, bbox } = detection;
      console.log(`Found object with ${confidence.toFixed(2)} confidence at ${bbox}`);
    });
    ```

## Best Practices

<Callout type="warning">
  For best results, ensure images are clear and well-lit. Consider using confidence thresholds to filter detections.
</Callout>

- Set appropriate confidence thresholds
- Process high-resolution images
- Consider object size and visibility
- Handle multiple detections appropriately

## Error Handling

<Tabs items={['Python', 'JavaScript', 'TypeScript']}>
  <Tabs.Tab>
    ```python
    try:
        response = await api.detect_object(
            image_url="https://example.com/image.jpg",
            object="person"
        )
    except InvalidImageError:
        print("Invalid image URL or format")
    except APIError as e:
        print(f"API error: {str(e)}")
    except Exception as e:
        print(f"Unexpected error: {str(e)}")
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```javascript
    try {
      const response = await api.detectObject({
        imageUrl: "https://example.com/image.jpg",
        object: "person"
      });
    } catch (error) {
      if (error instanceof InvalidImageError) {
        console.error("Invalid image URL or format");
      } else if (error instanceof APIError) {
        console.error(`API error: ${error.message}`);
      } else {
        console.error(`Unexpected error: ${error.message}`);
      }
    }
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```typescript
    try {
    ```
