## Getting Started

<Steps>
### 1. Install Dependencies
~~~bash
pip install transformers einops
~~~

### 2. Load and Use Model
<Callout type="info">
The model is updated regularly, so we recommend pinning the model version to a specific release as shown below.

Check out [Model Downloads](/model-downloads) for different model variants and their specifications.
</Callout>

~~~python
from transformers import AutoModelForCausalLM, AutoTokenizer
from PIL import Image

model_id = "vikhyatk/moondream2"
revision = "2024-08-26"
model = AutoModelForCausalLM.from_pretrained(
    model_id, trust_remote_code=True, revision=revision
)
tokenizer = AutoTokenizer.from_pretrained(model_id, revision=revision)

image = Image.open('<IMAGE_PATH>')
enc_image = model.encode_image(image)
print(model.answer_question(enc_image, "Describe this image.", tokenizer))
~~~
